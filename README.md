## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ 
- npm, yarn, pnpm, or bun

### Installation

1. Clone the repository:
```bash
git clone https://github.com/ErfanAalam/portfolio.git
cd portfolio
```

2. Install dependencies:
```bash
npm install
# or

pnpm install
```

```bash
pnpm dev
```
